PROYECTO FINAL - BASE DE DATOS DE COMICS

dentro de las carpetas de ForntendPj y backend se encuentran los archivos necesarios para exportar el proyecto de angular hacia otro dispositivo,
por cuestiones de espacio se omiten los archivos .angular y node_modules en el Frontend. 

Para este proyecto se utilizo maquetado mediante el paquete de bootstrap, por lo que se deberia instalar en la libreria mediante el comando 'npm install bootstrap',
y despues de haber instalado los paquetes agregar las referencias de esitlos en el arvhivo angular.json de la siguiente manera:

"styles": [
	"node_modules/bootstrap/dist/css/bootstrap.min.css",
	"src/stles.sass"
]
